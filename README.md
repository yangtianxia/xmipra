# xmipra

基于tarojs库创建小程序快速开发脚手架，支持微信、支付宝、抖音小程序，内置 `xmipra` API方法，`@xmipra/components` 开发组件，`@xmipra/plugins` tarojs插件库等。

## @xmipra/plugins

插件内置功能

- theme：主题配置插件
- env：环境插件
- project-config：小程序项目配置插件
- condition：快捷编译插件
- declaration：声明插件
